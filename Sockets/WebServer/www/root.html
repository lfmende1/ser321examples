You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /party? -- returns a page with the desired background and text color
  • /party? -- accessible colors: red, green, blue, yellow, orange, pink, purple
  • http://35.91.156.22:9000/party?bgr=pink&txt=orange
  • http://35.91.156.22:9000/party?bgr=red&txt=blue
  • /shapes? -- returns a page with the desired shape with desired dimensions
  • /shapes? -- accessible shapes: square, circle, triangle
  • http://35.91.156.22:9000/shapes?shape=triangle&size=150
  • http://35.91.156.22:9000/shapes?shape=circle&size=200
